ButtonGroup & ButtonToggle

Inform
ButtonGroup and ButtonToggle do not support uncontrolled versions.

ButtonGroup

Interface of ButtonGroup
üèó Coming Soon

ButtonGroup behaves like a checkbox group. Use this when the user may select zero or more of a small set of options. Control externally via value and onChange props. On individual ButtonItem, value is optional and will fall back to the text if absent.

Current selection: CA
;() => {
  const [value, setValue] = useState(['CA'])
  return (
    <>
      <ButtonGroup value={value} onChange={setValue}>
        <ButtonItem value="CA">California</ButtonItem>
        <ButtonItem value="AK">Alaska</ButtonItem>
        <ButtonItem value="UT">Utah</ButtonItem>
      </ButtonGroup>
      <Text pl="small">Current selection: {value.join(', ')}</Text>
    </>
  )
}

Disabled

<ButtonGroup value={['CA']} disabled>
  <ButtonItem value="CA">California</ButtonItem>
  <ButtonItem value="AK">Alaska</ButtonItem>
  <ButtonItem value="UT">Utah</ButtonItem>
</ButtonGroup>

ButtonToggle

Interface of ButtonGroup
üèó Coming Soon

ButtonToggle behaves as radio button group. Use this when the user may select only one of a small set of options. Control externally via value and onChange props. On individual ButtonItem, value is optional and will fall back to the text if absent.

Current selection: TypeScript
;() => {
  const [value, setValue] = useState('TypeScript')
  return (
    <>
      <ButtonToggle value={value} onChange={setValue}>
        <ButtonItem>Ruby</ButtonItem>
        <ButtonItem>TypeScript</ButtonItem>
        <ButtonItem>Python</ButtonItem>
      </ButtonToggle>
      <Text pl="small">Current selection: {value}</Text>
    </>
  )
}

Disabled

<ButtonToggle value="TypeScript" disabled>
  <ButtonItem>Ruby</ButtonItem>
  <ButtonItem>TypeScript</ButtonItem>
  <ButtonItem>Python</ButtonItem>
</ButtonToggle>

Nullable

Use the nullable prop if the user should be able to click the selected item to de-select it.

;() => {
  const [value, setValue] = useState('')
  return (
    <ButtonToggle value={value} onChange={setValue} nullable>
      <ButtonItem>Apples</ButtonItem>
      <ButtonItem>Oranges</ButtonItem>
      <ButtonItem>Bananas</ButtonItem>
    </ButtonToggle>
  )
}

Options

An alternative to composing a ButtonGroup or ButtonToggle with ButtonItem children is to pass it an array of options.

;() => {
  const options = [
    { value: 'Gouda', label: 'Smoked Gouda' },
    { value: 'Cheddar' },
    { value: 'Swiss', disabled: true },
  ]
  const [value, setValue] = useState([options[0].value])
  return <ButtonGroup options={options} value={value} onChange={setValue} />
}
